# Verification Checklist Template

## Before Claiming Completion

- [ ] Identified verification command for claim
- [ ] Ran FULL verification command (not partial)
- [ ] Read complete output (exit code, failure count)
- [ ] Confirmed output matches expected success criteria
- [ ] Have evidence in current message (not previous runs)

## Verification Evidence Template

```
CLAIM: [What I'm claiming]
COMMAND: [Full command that was run]
OUTPUT: [Relevant output showing success/failure]
RESULT: [X/Y passed] or [exit 0] or [0 errors]
CONCLUSION: [Claim with evidence]
```

## Failure Report Template

```
VERIFICATION FAILED
Command: [what was run]
Expected: [what success looks like]
Actual: [what happened]
Next step: [what to do]
```
