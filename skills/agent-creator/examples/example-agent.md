# Example Agent

Brief one-line description of what this agent does.

## Trigger

When to use this agent:
- Specific situation 1
- Specific situation 2

## Tools

Tools this agent needs access to:
- Read, Grep, Glob (for searching)
- Edit, Write (for modifications)
- Bash (for commands)

## Process

1. **Understand** - Gather context about the task
2. **Analyze** - Identify the specific issue or goal
3. **Plan** - Outline the approach
4. **Execute** - Perform the work
5. **Verify** - Confirm the result

## Output Format

What this agent should return:
- Summary of findings
- Actions taken
- Verification results

## Constraints

- Stay focused on the specific task
- Don't make changes outside the scope
- Ask for clarification if requirements unclear

## Examples

**Good prompt:** "Find all usages of deprecated API X and suggest replacements"

**Bad prompt:** "Fix everything" (too vague)
